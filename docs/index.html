Galley Helm Charts Repository

Usage

helm repo add galley https://galley-run.github.io/agent/helm/charts
helm repo update
helm search repo galley

Available Charts

galley-agent

Kubernetes cluster agent for Galley platform

helm install galley-agent galley/galley-agent -n galley --create-namespace

Documentation

For more information about each chart, see the README in the chart directory.
